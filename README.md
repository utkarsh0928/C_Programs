# Basic C Programs Collection

A collection of beginner-level C programs covering fundamental concepts.

## Topics Covered
- **Arrays:** Sorting, searching, and matrix operations.
- **Functions:** Modular programming examples.
- **Recursion:** Factorial, Fibonacci series.
- **Patterns:** Star and number printing patterns.
- **Pointers:** Basic memory management examples.

You can compile these files using GCC:

gcc filename.c -o output
